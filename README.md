# About my first project on GitHub.
Това е проект тип социална мрежа, който следи прогреса на потребителите в тренировките и храненето чрез календар, в който може да се попълват различни въпроси. Платформата позволява на потребителите да цели събития, които биха им били интересни и полезни. Всеки може да качва съдържание, като идеята е да бъде полезно и свързано със здравословен начин на живот. В бъдещата разработка предстои добавяне на свързаност между потребителите чрез система за съобщения.

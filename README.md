# About my first project on GitHub.
Това е проект тип социална мрежа, която следи прогреса на потребителя в тренировките и храненето чрез календар, в който могат да се попълват различни въпроси и дневни записи.
Платформата позволява на потребителите да откриват събития, които биха им помогнали според техните цели. Всеки може да качва съдържание, като идеята е то да бъде полезно и свързано със здравословен и активен начин на живот. Има опция за запазване на публикации.
В началната страница („Index“) има късмети, тестове и забавни елементи, които правят използването по-интересно.
В бъдещата разработка предстои добавяне на
1. класация за най-активните потребители;
2. система за съобщения между тях
3. още интерактивни функции.
   
Благодаря, че отделяте време да разгледате проекта ми!
